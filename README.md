Automation
=============

## Setup
* Install python3
* Install chromedriver
* Install required python libs: robotframework, selenium

### For Mac OS X

```
brew install python3
brew cask install chromedriver
pip3 install -r requirements.txt
```

## Run
```
robot --pythonpath services features
```
